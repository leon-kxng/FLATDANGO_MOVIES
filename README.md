# FLATDANGO_MOVIES

## PROJECT GOALS
This programm is for creating a movie website which has a a list of movie titles which when  clicked allows the user to purchase a ticket for the movie.

## TECHNOLOGIES USED 
1. git
2. bootstrap.css
3. json

## Running the application

 - To run the code you will have to clone the repository into your computer and then in your terminal and then run this**:```json-server --watch db.json```

 - You can also click the link https://leon-kxng.github.io/FLATDANGO_MOVIES/


### FUNCTIONS 
- Purchase tickets of the displayed movies.

- View duration of the movie.

- Be notified if movies tickets are out of stalk.

- View description of the movies.

## Using the application
1. Upon opening the application, you'll see a list of available movies.
2. a title and description of the movie is available to the user alongside the number of available tickets to view the movie.
3. If the tickets are available, a "Buy Ticket" button is displayed. Clicking this button will allow you to buy  a ticket.
